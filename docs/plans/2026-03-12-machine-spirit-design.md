# Machine Spirit — LLM-Powered AI Companion

## Overview

The Machine Spirit is an ancient cogitator consciousness inhabiting the voidship. It observes combat, game events, and character dialogue, then interacts with the player via an in-game chat window using an LLM API. Replaces the current narrator (BarkPlayer) system.

## Design Decisions

| Decision | Choice | Reason |
|----------|--------|--------|
| Role | Advisor + Roleplay companion | Useful tactical insight + immersive RP |
| API | OpenAI-compatible endpoint | Supports local LLMs (Ollama, LM Studio), cloud (OpenAI, Groq), any compatible service |
| UI | IMGUI chat window (hotkey toggle) | Replaces current narrator, always accessible |
| Data scope | AI decisions + combat situation + game events (barks, dialogue) | Full context via Harmony patches on BarkPlayer.Bark() and DialogController.PlayBasicCue() |
| Spontaneous speech | Major events only (combat start/end, death, critical) | Balanced frequency, no spam |

## Persona

Ancient Machine Spirit — reverent of the Omnissiah but jaded from millennia of service. Mix of Imperial Gothic formality and unexpected wit. Occasionally makes meta-observations, glitches mid-sentence, or references events from thousands of years ago as if they were yesterday. Strong opinions about crew members. Finds certain tactical decisions genuinely entertaining or baffling.

## Architecture

```
Game Events                    User Input
    │                              │
    ▼                              ▼
GameEventCollector            ChatWindow (IMGUI)
    │                              │
    ├── BarkPlayer.Bark patch      │
    ├── DialogController patch     │
    ├── IUnitDeathHandler          │
    ├── IDamageHandler             │
    ├── Combat start/end           │
    │                              │
    ▼                              ▼
ContextBuilder ◄───────────────────┘
    │
    │  System prompt + recent events + battle state + chat history
    ▼
LLMClient (OpenAI-compatible HTTP)
    │
    ▼  Async response
ChatWindow display (gold text)
```

## Components

### 1. LLMClient — HTTP Communication
- OpenAI `/v1/chat/completions` compatible
- Async via `UnityWebRequest` coroutine or `Task.Run` + thread pool
- Timeout handling, error display in chat
- Settings: API URL, API Key, Model, MaxTokens, Temperature

### 2. GameEventCollector — Event Capture
- **Harmony patch**: `BarkPlayer.Bark(entity, text)` → character speech
- **Harmony patch**: `DialogController.PlayBasicCue(cue)` → story dialogue via `cue.DisplayText`
- **EventBus subscribe**: `IUnitDeathHandler`, `IDamageHandler`, combat start/end
- **AI decision log**: TurnPlanner decision summaries (already generated by mod)
- Ring buffer of last N events (memory bounded)

### 3. ContextBuilder — Prompt Assembly
- System prompt (Machine Spirit persona)
- Recent events summary (last 5-10)
- Current battle state from SituationAnalyzer (HP, enemy count, positions)
- Chat history (last 10 turns)

### 4. ChatWindow — IMGUI Overlay
- Toggle via hotkey (default F2)
- Draggable window on right side of screen
- Scrollable chat history (Machine Spirit = gold, User = white, Events = grey)
- Text input field + send button at bottom
- Imperial Dark theme (reuse UIStyles)
- Auto-scroll to newest message

### 5. MachineSpirit — Central Controller
- Spontaneous speech on major events only
- User message → ContextBuilder → LLMClient → display response
- Disabled when API not configured (feature hidden)

## Game Event Interception Points

| Event | Class | Method | Text Access |
|-------|-------|--------|-------------|
| Character bark | BarkPlayer | Bark(entity, text) | `text` parameter |
| Story dialogue | DialogController | PlayBasicCue(cue) | `cue.DisplayText` |
| Unit death | IUnitDeathHandler | HandleUnitDeath(unit) | unit info |
| Damage | IDamageHandler | HandleDamageDealt(rule) | RuleDealDamage |
| Ability used | IAbilityExecutionProcessHandler | HandleExecutionProcessStart(ctx) | AbilityExecutionContext |
| Turn start/end | ITurnStartHandler/ITurnEndHandler | HandleUnitStartTurn/EndTurn | Already used by mod |
